# TaskFlow

TaskFlow is a Task Management System built using ASP.NET Web API, C#, and Entity Framework Core.
It provides a RESTful API for managing tasks, users, and projects, allowing for efficient task tracking
and collaboration with team members.

# Design Patterns

- Repository Patttern
- Unit of Work Pattern
- Strategy Pattern (Console Application)
- Automapper
